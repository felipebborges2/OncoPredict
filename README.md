# OncoPredict-ML

Machine learning pipeline for breast cancer prediction using clinical features.

This project was developed as part of my portfolio at the intersection of **Medical Physics**, **Oncology**, and **Artificial Intelligence**. It explores a classic breast cancer dataset and compares machine learning models for classification, including Logistic Regression, Random Forest, and Support Vector Machine (SVM).

## Project Goals

- Build a complete end-to-end machine learning pipeline
- Explore clinical-style tabular data
- Compare different classification models
- Evaluate performance using relevant metrics
- Save the best trained model for inference
